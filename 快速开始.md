# Xobi 快速开始指南

## 环境要求

1. **Node.js** 18+ - [下载地址](https://nodejs.org/)
2. **Python** 3.10+ - [下载地址](https://www.python.org/)

## 安装步骤

### 1. 安装依赖
双击运行 `安装依赖.bat`，等待安装完成。

### 2. 配置 API Key
1. 复制 `xobixiangqing/.env.example` 为 `xobixiangqing/.env`
2. 编辑 `.env` 文件，填写你的 API Key：
   ```
   OPENAI_API_KEY=你的API密钥
   ```

### 3. 启动服务
双击运行 `Xobi启动器.bat`

### 4. 访问应用
打开浏览器访问：http://localhost:3000

## 常见问题

**Q: 安装依赖失败？**
- 确保 Node.js 和 Python 已正确安装
- 尝试以管理员身份运行

**Q: 启动失败？**
- 检查端口 3000、5000 是否被占用
- 检查 .env 文件是否正确配置

**Q: API 调用失败？**
- 确认 API Key 正确
- 检查网络连接
